What is the Document Object model (DOM) ?

(DOM) is a programming interface for web documents.allowing dynamic manipulation of content, styles, and structure.

How do you select un Element from (DOM) ?

To select an element from the DOM, use JavaScript methods like getElementById(), querySelector(), or getElementsByClassName(), targeting specific IDs, classes, or tags within the document.

What is event delegation in the context of the (DOM),and why is it useful .

Event delegation in the DOM allows a parent element to handle events for its child elements. It improves efficiency by reducing event listeners and enabling dynamic content handling.

How do you manipulate element's attributes and uses style the (DOM)?

To manipulate an element's attributes in the DOM, use setAttribute() or removeAttribute(). For styling, modify the style property directly, e.g., element.style.color = "blue";.
